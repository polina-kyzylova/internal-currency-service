# Сервис внутренней валюты
Это система корпоративной мотивации сотрудников, которая позволят контролировать общий объем средств, управлять финансовыми потоками и ЦФО (центр финансового обеспечения), а также обеспечивать обмен внутренней валюты.

Категории пользователей:
- **Сотрудник** - переводит валюту коллегам и отслеживает личный баланс
- **Владелец ЦФО** - анализируют эффективность команд и распределяет поощрения из своего ЦФО
- **Администратор** - контролирует бюджет и управляет всеми ЦФО

## Architecture

**Key points**
1. Atomic Design - модульная компонентная архитектура для масштабирования и переиспользования кода
2. Adaptive UI - адаптивный интерфейс для всех типов устройств
3. Form validation - валидация пользовательского ввода для предотвращения ошибок и некорректных транзакций
4. JWT-authentication - авторизация с использованием access/refresh токенов и автоматическим продлением сессии
5. Role-Based routing - система приватных маршрутов для контроля доступа разных категорий пользователей
6. Real-Time updates - использование Long-Polling для мгновенного отображения изменений баланса

**Tech Stack**
- React
- RTK Query
- Redux
- Material UI
- CSS modules
- React Hook Form
- React Router DOM
- Figma
