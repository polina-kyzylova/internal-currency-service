# Сервис внутренней валюты
Это система корпоративной мотивации сотрудников, которая позволят контролировать общий объем средств, управлять финансовыми потоками и центрами затрат, а также обеспечивать обмен внутренней валюты.

Категории пользователей:
- **Администратор** - контролирует бюджет и управляет всеми ЦФО (центр финансового обеспечения)
- **Владелец ЦФО** - анализируют эффективность команд и распределяет поощрения из своего ЦФО
- **Сотрудник** - переводит валюту коллегам и отслеживает личный баланс


## About
Проект реализован в рамках ИТ-лагеря Холдинга Т1 — интенсива с реальными бизнес-кейсами под менторством экспертов компании.
